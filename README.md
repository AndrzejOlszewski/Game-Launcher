# Python Game Launcher 🎮

A custom, text-based game launcher built in Python that simulates network connections and updates.

## Features
* **Dynamic Choices**: Supports Fortnite, Minecraft, and Roblox.
* **Realistic Delays**: Uses the `time` module to simulate server connections.
* **Retro Countdown**: Features a custom-coded `100% to 0%` download timer loop.

## How to Run
1. Make sure you have Python installed.
2. Run the script in your terminal: `python launcher.py`
